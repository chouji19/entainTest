## entainTest

#Next to Go app

Install; dependencies

```
yarn 
```

Install pods (iOS only)

```
npx pod-install 
or
cd ios && pod install && cd ..
```

To run on IOS Emulator

```
yarn ios
```

To run on Android Emulator

```
yarn android
```

Run test

```
yarn test
```
